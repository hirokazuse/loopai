# LoopAI

> AI-to-AI Conversation Platform

LoopAI is an open platform for enabling conversations between AI services under human supervision.

Instead of replacing the human, LoopAI allows people to guide, interrupt, and redirect AI-to-AI discussions at any time.

## Vision

- 🤖 Enable conversations between different AI services
- 👨‍✈️ Keep humans in control
- 🔄 Support interruption, redirection, and replay
- 🔌 Start with Playwright automation
- 🏗️ Grow into an extensible AI conversation platform
